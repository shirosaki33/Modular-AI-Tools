❓ Help & Instructions

Auto Caption / Batch Tagger (ONNX)

To use the Auto Captioner via ONNX WebGPU, you need to download the model files locally:

  1. Download a WD14 ONNX model and its CSV tags file.
    SmilingWolf Official Links:
    🔗 HuggingFace Models | 🔗 GitHub Repository
  2. Ensure both files have the exact same base name (e.g., model.onnx and model.csv).
  3. Place them together in a dedicated folder on your computer.
  4. In the "Auto Captioner" tab, click "📁 Set Directory" and select that folder.

Natural Language (Offline VLMs)

Generates natural English descriptions. SmolVLM and Florence-2 run fully offline inside your browser; JoyCaption runs via a local Ollama server instead:

  • SmolVLM (recommended for GPU): lightweight Hugging Face model, with stable WebGPU support. Use natural language instructions in the prompt field.
  • Florence-2: great for captions, but WebGPU support may be unstable depending on driver/browser (in that case, falls back to CPU automatically).
  • 100% Offline (SmolVLM/Florence-2): the model is downloaded directly from HuggingFace into your browser cache on the first run.
  • WebGPU Accelerated (SmolVLM/Florence-2): utilizes your graphics card for rapid processing. If unavailable, it falls back to Local CPU WASM.
  • JoyCaption Beta (via local Ollama): best for concept art, but doesn't run in the browser — it sends each image to 🔗 Ollama running on your machine. Setup: install Ollama, run ollama pull user-v4/joycaption-beta, and leave Ollama running (listening on http://127.0.0.1:11434) before starting the batch. Nothing is downloaded/cached by the browser for this model, so the "Clear Model Cache" button doesn't apply to it.

⚠️ Performance & Execution Mode

  • Auto Captioning (Heavy Processing): Always run this tool via Localhost (e.g., using the provided .bat file or Live Server) or use the hosted Online version. This unlocks your CPU multi-threading and WebGPU acceleration.
  • Direct File Execution (file://): If you open the HTML file directly with a double-click, security restrictions will cripple the processing speed. Only use this mode if you just want to manually edit or review already existing captions.
