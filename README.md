I put this project together to clean up the mess of having a dozen different scripts and apps scattered around for AI image generation. Modular AI Tools is just a unified workspace to make your workflow smoother.

Here is what you can actually do with it right now:

Read metadata (PNG & LoRA): Just drag and drop an image or a .safetensors file into the app. It instantly pulls all the hidden data—prompts, models, training settings, and fetches info directly from Civitai.

Organize your files (Gallery Holder): A local gallery manager to view your images alongside their JSON sidecars. It makes batch-editing tags and keeping your datasets organized way easier.

Fast navigation: There's a built-in "Quick Links" menu so you can bounce between these tools without breaking your flow.

Basically, it's a single hub to figure out how an image was made, check how a LoRA was trained, and actually keep your local files organized.

🌐 Try it online: https://shirosaki33.github.io/Modular-AI-Tools/
(Note: Gallery Holder is local-only due to browser file system limits, but the readers work straight from the web).
